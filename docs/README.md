# Dokumentindeks

## Status

Aktiv baseline er `fc4552c`. Denne indeksen skiller mellom dokumenter som beskriver implementert baseline, planlagt retning og historisk referanse.

## Dokumenter I Denne Oppgaven

### `../README.md`

- Formål: kort prosjektintroduksjon og startpunkt for repoet.
- Status: oppdatert mot baseline `fc4552c`.
- Bør leses av: alle som skal inn i prosjektet.
- Beskriver: både implementert baseline og nåværende prosjektfase.

### `APP_STORE_READINESS.md`

- Formål: intern kravliste for senere innsending til App Store.
- Status: plan og verifikasjonsliste, ikke lanseringsklar.
- Bør leses av: utviklere, designere og den som forbereder review.
- Beskriver: planlagte krav og verifisering, ikke ferdig funksjonalitet.

### `DESIGN_SYSTEM.md`

- Formål: felles designretning og tokens.
- Status: blanding av etablert baseline og planlagt videreføring.
- Bør leses av: designere og utviklere før ny skjerm bygges.
- Beskriver: både etablert stil og planlagt systemarbeid.

### `NEW_CHAT_PROMPT.md`

- Formål: ferdig prompt for å fortsette arbeidet i ny chat.
- Status: operativ hjelptekst.
- Bør leses av: den som overtar arbeidet i ny samtale.
- Beskriver: planlagt arbeidsmåte og låste beslutninger.

### `PRODUCT_DECISIONS.md`

- Formål: beslutningslogg for produkt og retning.
- Status: aktiv referanse for hva som er besluttet, planlagt og åpent.
- Bør leses av: produkt, design og utvikling.
- Beskriver: planlagt retning og låste valg, ikke full implementasjon.

### `TREASURE_HUNT_HANDOFF.md`

- Formål: presis overlevering for skattejakt-arbeidet.
- Status: baseline-håndoff med tydelig skille mellom kode og plan.
- Bør leses av: neste person som jobber med skattejakt.
- Beskriver: faktisk baseline og planlagt videre retning.
