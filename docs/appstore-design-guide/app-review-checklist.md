# App Review-sjekkliste

## Stabilitet og funksjon

- [ ] Appen er testet på fysisk iPhone og iPad der det er relevant.
- [ ] Ingen krasj, blokkerte flyter eller døde lenker.
- [ ] Backend og testdata er tilgjengelig under review.
- [ ] Demo-konto eller demo-modus er dokumentert når innlogging kreves.
- [ ] Ingen placeholder-tekst, tomme skjermer eller midlertidige elementer.

## Design og tilgjengelighet

- [ ] Interaktive kontroller har minst 44 × 44 pt trykkflate.
- [ ] Vanlig tekst har minst 4,5:1 kontrast.
- [ ] Store tekster og ikke-tekstlige kontroller har minst 3:1 kontrast.
- [ ] Tekst bruker systemfont og støtter skalering.
- [ ] Safe areas og fleksibel layout er brukt.
- [ ] Valgt, feil og suksess kommuniseres med mer enn bare farge.
- [ ] Interaktive elementer har tilgjengelighetsetiketter.
- [ ] Fokusrekkefølge og VoiceOver er kontrollert.

## Personvern og tillatelser

- [ ] Personvernerklæring finnes i App Store Connect og i appen.
- [ ] Konto kan slettes fra appen dersom konto kan opprettes.
- [ ] Tillatelser er nødvendige og har konkrete purpose strings.
- [ ] Appens personverndata i App Store Connect er korrekte.

## Metadata og innsending

- [ ] Skjermbilder viser faktisk bruk av appen.
- [ ] Beskrivelse, nøkkelord og aldersgrense er korrekte.
- [ ] Review Notes forklarer ikke-opplagte funksjoner.
- [ ] Kjøp, abonnement og tilgangsbegrensninger er tydelig beskrevet.
- [ ] Versjonsnummer, build og release notes er kontrollert.

## Live Rebus-designstandard

- [ ] 8-punkts spacing-grid med 4-punkts mikrospacing.
- [ ] Horisontal margin 16 pt på kompakte skjermer.
- [ ] Primærknapp minst 48 pt høy, helst 52 pt.
- [ ] Systemfont brukes uten hardkodet fontfil.
- [ ] Skjermbilder er visuelt kontrollert mot godkjent referanse.
